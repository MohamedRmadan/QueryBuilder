# QueryBuilder

A Small project to filter users and show some chars.

## Installation

### Backend 
using npm

```bash
npm install
```
### Front end 
using npm

```bash
cd query-builder-ui
npm install
```

## Start project
in the main folder run the following command after finishing installation

```python
npm run dev
```

## After Start

### App should be ready
### front end link :
```link
http://localhost:3000
```
### back end link :
```link
http://localhost:5000
```
